# План реализации (утвержден пользователем)

## 1) Информация, которую получили
- Проект создаётся с нуля.
- 2D RTS: редактор карты (с превью) + игра запускаются отдельно.
- Формат сохранения: **JSON**.
- Динамический фронт: каждый юнит имеет до **10 “очков”**; при приближении юнита к линии фронта фронт смещается. Если соотношение врагов **> 2 раза** или очки превышают — фронт смещается в обратную сторону.

## 2) План по файлам
### editor/
- `editor_main.py` — окно редактора (pygame), превью правой/нижней панели (arcade/pygame).
- `map_model.py` — общая структура данных карты для editor+game.
- `frontline.py` — модель/алгоритм фронта на основе “очков” юнитов.
- `storage.py` — save/load карты в JSON.
- `editor_tools.py` — размещение/выделение/перетаскивание/удаление объектов.

### game/
- `game_main.py` — загрузка карты, RTS-цикл, рендер.
- `entities.py` — сущности: `Unit`, `Building`, `Barrier`, `Wall`, `Landscape`.
- `movement.py` — плавное движение (не тиковое по тайлам), коллизии с препятствиями.
- `selection_control.py` — выделение и приказы.

### maps/
- `example_139x189.json` — тестовая карта.

## 3) Зависимости
- `pygame`
- `arcade`
